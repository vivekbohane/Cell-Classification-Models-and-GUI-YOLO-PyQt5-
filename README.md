# Cell_Classification_Models_and_GUI_Using_YOLO_and_PyQt5
 Cell classification models were trained using YOLOv8 and a GUI was built using PyQt5.
 
# Requirements:

1. PyQt5
2. ultralytics - YOLO
3. cv2 (OpenCV)
4. collections
5. sys
